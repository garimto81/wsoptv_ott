MVP LAYOUT WIREFRAME

Three-Tier Layout Structure

The platform follows the NBA TV style with a three-tier layout structure optimized for sports content discovery and viewing.

Figure 1: Three-tier layout structure for the OTT platform

Layout Components

  1. Live Ticker (Top) - Horizontal tournament status bar displaying live events, similar to NBA TV's score strip. Provides instant access to ongoing tournaments.
  2. Video Player (Middle) - 16:9 aspect ratio with Timeshift capability. Features live/VOD playback with chapter navigation and stream selection.
  3. Content Section (Bottom) - Highlights, categories, and related content organized by series, events, and player focus.

Design Philosophy: Minimize clicks to content. Users should reach live streams or VOD content within 2-3 clicks from any page.
