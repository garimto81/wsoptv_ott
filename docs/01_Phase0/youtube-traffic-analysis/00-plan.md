## 1. 배경 및 문제 정의

### 왜 YouTube 분석인가

WSOPTV는 Q3 2026 론칭 예정인 Vimeo SaaS 기반 포커 OTT 플랫폼이다. 론칭 전 트래픽 예측이 필요한 이유:

| 필요성 | 설명 |
|--------|------|
| **Vimeo 대역폭 계약** | 월간 예상 시청 시간에 따라 CDN 비용 및 Vimeo 플랜 결정 |
| **수익 모델 검증** | 구독자 수 예측 -> 매출 시나리오 -> 투자 정당성 확보 |
| **콘텐츠 전략 수립** | 어떤 콘텐츠가 시청자를 끌어오는지 데이터 기반 판단 |
| **마케팅 예산 산정** | CAC 추정 근거 |
| **경쟁사 벤치마크** | PokerGO, PokerStars 대비 WSOPTV 포지셔닝 |

### 기존 문서와의 관계

| 기존 문서 | 범위 | 본 분석의 추가 가치 |
|-----------|------|---------------------|
| `05-competitor-ott-analysis` | NFL/MLB/ESPN+ OTT 플랫폼 기능/가격 비교 | YouTube 시청 데이터 기반 수요 정량화 추가 |
| `06-ott-solutions-analysis` | 14개 OTT 솔루션 벤더 비교 + PokerGO $7.5M 매출 | YouTube 채널 데이터로 전환율 모델 추가 |
| `07-nbatv-complete-analysis` | NBA TV/League Pass 전략 분석, 구독자 3-5M 추정 | 포커 시청자 데이터 + YouTube 차원 분석 추가 |
| `01-mvp-spec` | Vimeo 기반 MVP 스펙 | 예상 트래픽 수치 제공 |

**본 분석의 고유 범위**: YouTube 공개 데이터를 기반으로 포커 콘텐츠의 실제 수요를 정량화하고, WSOPTV 론칭 시 예상 트래픽과 구독자 수를 산출한다. **기존 문서에 이미 존재하는 구독자/매출 데이터를 재추정하지 않고, YouTube 채널 메트릭이라는 새로운 데이터 차원만 추가한다.**

### 데이터 재사용 원칙

기존 문서와의 수치 불일치를 방지하기 위해, 아래 데이터는 기존 문서에서 직접 참조한다:

| 데이터 | 출처 문서 | 값 | 본 분석에서의 용도 |
|--------|----------|-----|-------------------|
| NBA League Pass 구독자 | `07-nbatv` 7.2절 | 3-5M (추정, +-30%) | Tier 3 벤치마크 비교 |
| PokerGO 연매출 | `06-ott-solutions` 벤치마크절 | $7.5M | 역산 모델 앵커 |
| PokerGO 구독 가격 | `06-ott-solutions` 벤치마크절 | $14.99/월, $99.99/년 | ARPU 계산 |

> **규칙**: 위 데이터를 본 분석에서 재추정하지 않는다. 기존 문서의 수치를 그대로 사용하고 출처를 명기한다.

2. 채널 성장 추세 파악에 12개월이면 충분
3. 더 긴 추이가 필요한 경우 stats.video/Noxinfluencer의 연간 그래프로 보완
4. YouTube 채널 About 페이지의 "가입일, 총 조회수"로 장기 평균 역산 가능

**수집 절차**:
1. Social Blade에서 채널 검색
2. Monthly Statistics 수집 (최근 12개월)
3. Yearly Statistics 수집 (연도별 합산 데이터 — 무료 제공)
4. 스크린샷 캡처 + 수치 테이블화

## 4. 분석 방법론

### 4.1 TAM/SAM/SOM 추정 모델

YouTube 데이터에서 시장 규모를 역산하는 프레임워크:

```
TAM (Total Addressable Market)
+-- 전 세계 포커 YouTube 시청자 (unique viewers 추정)
+-- 근거: 주요 포커 채널 구독자 합산 - 중복 보정
+-- 보정 계수: 구독자 1명 = 비구독 시청자 3-5명 (YouTube 평균)

SAM (Serviceable Addressable Market)
+-- 포커 OTT 유료 구독 의향 시청자
+-- 근거: YouTube engagement rate > 평균인 시청자
+-- 보정 계수: 높은 engagement = 유료 전환 가능성

SOM (Serviceable Obtainable Market)
+-- WSOPTV 론칭 첫해 실제 획득 가능 구독자
+-- 근거: PokerGO 벤치마크 + WSOP 브랜드 가치 보정
+-- 보정 계수: 브랜드 인지도, 가격, 콘텐츠 차별성
```

### 4.2 전환율 모델

#### Primary Model: 단일 비율 벤치마크 모델

**모델 구조**: YouTube 구독자 -> OTT 유료 구독자 직접 전환 비율

**앵커: PokerGO 역산**

```
PokerGO 연매출: ~$7.5M (출처: 06-ott-solutions 벤치마크절)
구독 가격: $14.99/월, $99.99/년 (출처: 06-ott-solutions 벤치마크절)

ARPU 다중 시나리오:

  시나리오 A: 평균 구독 3개월 (높은 churn 가정)
    ARPU = $14.99 x 3 = ~$45/년
    구독자 = $7.5M / $45 = ~167K명

  시나리오 B: 평균 구독 6개월 (중간 가정)
    ARPU = $14.99 x 6 = ~$90/년
    구독자 = $7.5M / $90 = ~83K명

  시나리오 C: 평균 구독 9개월 / 연간 플랜 혼합 (낮은 churn 가정)
    ARPU = ($14.99 x 9 x 0.5) + ($99.99 x 0.5) = ~$117/년
    구독자 = $7.5M / $117 = ~64K명

  --> 구독자 범위: 64K ~ 167K명
  --> 중앙 추정: ~83K명 (6개월 기준)

YouTube 구독자: 694,000명

YouTube 구독자 대비 유료 구독자 비율:
  하한: 64K / 694K = ~9%
  중앙: 83K / 694K = ~12%
  상한: 167K / 694K = ~24%

단, YouTube 구독자 =/= 전체 유입 경로
YouTube 직접 전환 추정: 5-8% (유입 경로 다양성 보정)
```

> **참고**: v1.0에서 "평균 구독 기간 6개월"을 단일 가정으로 사용한 것을 3/6/9개월 다중 시나리오로 확장했다. 6개월 가정의 근거는 스포츠 OTT의 일반적 시즌 기반 구독 패턴(시즌 시작에 가입, 시즌 종료 후 해지)이며, 이는 포커의 WSOP 시즌(5-7월, 약 3개월)과 비시즌 구독 유지율에 따라 달라진다. 정확한 churn 데이터가 없으므로 3개 시나리오 모두 제시한다.

**3-시나리오 (WSOPTV)**:

| 시나리오 | YouTube 전환 비율 | 근거 |
|----------|:-----------------:|------|
| **보수적** | 3-5% | PokerGO 하한보다 보수적. WSOP 브랜드 인지도가 높으나 OTT 신규 진입 |
| **기본** | 5-8% | PokerGO 역산 직접 전환율과 동일. WSOP 브랜드 + GGPoker 시너지 |
| **낙관적** | 8-12% | PokerGO 전체 전환율에 근접. 풀 마케팅 + 독점 콘텐츠 + 인플루언서 |

#### Secondary Model (부록): 5단계 전환 퍼널 (설명용)

> 5단계 퍼널 모델은 `04-conversion-model.md`의 Appendix A에 보존한다. 각 단계의 전환율 범위를 곱하면 0.003%-0.54%로 180배 차이가 나므로, 의사결정 근거로는 부적합하다. 단, 각 단계의 개선 레버를 식별하는 데 유용하므로 설명 자료로 유지한다.

```
YouTube 구독자 -> 월간 활성 시청자 (15-40%)
  -> OTT 인지 사용자 (30-60%)
  -> 무료 체험 가입 (5-25%)
  -> 유료 전환 (3-12%)
  -> 12개월 잔존 (50-75%)

결합 범위: 0.003% ~ 0.54% (180x 차이)
--> 의사결정용 부적합, 레버 식별용으로만 사용
```

### 4.5 월간 시청 시간 모델

```
월간 스트리밍 시간 = 활성 구독자 x 구독자당 월평균 시청 시간

구독자당 월평균 시청 시간 추정 (업계 추정, LOW 신뢰도):
  WSOP 시즌 (5-7월): 15-25시간/월 (라이브 이벤트 집중)
  비시즌 (8-4월): 3-8시간/월 (VOD, 아카이브)
  연간 가중 평균: ~7-12시간/월

벤치마크 (업계 추정, LOW 신뢰도):
  NBA League Pass: 구독자당 ~12시간/월 (시즌 중)
  Netflix 일반: ~3.2시간/일 -> ~96시간/월 (비교 불가, 참고만)
  스포츠 OTT 평균: ~8-15시간/월 (시즌 중)

Vimeo CDN 비용 계산 예시:
  기본 시나리오: 20K 활성 구독자 x 10시간/월 = 200K 시간/월
  1080p 비트레이트: ~5 Mbps = ~2.25 GB/시간
  월간 대역폭: 200K x 2.25GB = ~450 TB/월
```

### 4.7 콘텐츠 유형별 성과 분석

YouTube 인기 콘텐츠 유형과 WSOPTV 콘텐츠 전략의 매핑:

### 4.8 시즌성 분석 방법

```
월별 조회수 데이터 (12개월 + 연간 합산)
    |
    +-- WSOP 시즌 (5-7월) 평균 조회수
    +-- 비시즌 (8-4월) 평균 조회수
    |
    +-- 시즌 승수 = 시즌 평균 / 비시즌 평균
    |   예상: 2.5-4x
    |
    +-- WSOPTV 시사점:
        +-- 피크 트래픽 = 연간 평균 x 시즌 승수
        +-- Vimeo CDN 피크 대역폭 계산 근거
        +-- 비시즌 콘텐츠 전략 필요성 정량화
```
