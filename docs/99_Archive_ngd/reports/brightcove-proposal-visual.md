## 3. 제공 기능 (Phase 1)

![Features](../images/brightcove-proposal/13-features.png)

### Screens
- Dynamic Landing Page
- Movie/Show/Program Detail
- Sports Detail Page
- Search Screen
- Program Guide (EPG)
- My Content

### User Journeys
- Sign In / Sign Up
- Payment (SVOD + TVOD)
- Favourites / Bookmarks
- Profile Management
- Social Sharing

### Playback
- Full Screen DRM Players
- VOD / Live
- Binge Watch / Autoplay
- Chromecast
- **Multiview Experience**
