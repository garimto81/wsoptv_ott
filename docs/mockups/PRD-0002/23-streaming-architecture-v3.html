SECTION D: Transcoding & Packaging (3 Nodes)
NODE 15 ABR Transcoding
WHAT Adaptive Bitrate Transcoder
ROLE Convert single master to multiple resolutions/bitrates (1080p~240p, 8~0.3Mbps)
INPUT Final Master Video (ProRes/H.264 Mezzanine)
OUTPUT ABR Ladder: HLS/DASH segments (Clear, unencrypted)
-> NEXT DRM Packaging
|
Internal Pipeline HLS/DASH (Clear) 0.3-8 Mbps
NODE 16 DRM Packaging
WHAT DRM Encryption Packaging System
ROLE Encrypt content with Widevine/FairPlay/PlayReady DRM, integrate license key issuance
INPUT Clear (unencrypted) HLS/DASH segments
OUTPUT CENC/SAMPLE-AES encrypted segments + DRM Manifest
-> NEXT Origin Server
|
S3 PUT Encrypted fMP4 + Manifest Variable
NODE 17 Origin Server
WHAT Origin Storage Server (S3 + Origin Shield)
ROLE Store encrypted content, serve origin to YouTube RTMP and WSOPTV CDN
INPUT DRM packaged HLS/DASH content
OUTPUT YouTube RTMP (Branch 1) + WSOPTV CDN Pull (Branch 2)
-> NEXT YouTube RTMP Ingest / CDN Ingest (SPLIT)
