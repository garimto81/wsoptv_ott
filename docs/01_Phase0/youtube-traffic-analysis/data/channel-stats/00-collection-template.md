## 4. 콘텐츠 태그 정의

| 태그 | 정의 | 예시 |
|------|------|------|
| **LIVE** | 라이브 스트리밍 방송 | WSOP Main Event 라이브 |
| **HIGHLIGHT** | 하이라이트 편집 영상 (5-15분) | Final Table 명장면 모음 |
| **FULL_EVENT** | 풀 이벤트 녹화 (30분+) | Day 1A 전체 방송 |
| **VLOG** | 플레이어 Vlog/일상 | Brad Owen 라스베가스 Vlog |
| **TUTORIAL** | 전략/교육 콘텐츠 | How to Play AK Preflop |
| **ANALYSIS** | 전문가 해설/분석 | Daniel Negreanu 핸드 리뷰 |
| **INTERVIEW** | 인터뷰/대담 | Phil Hellmuth 인터뷰 |
| **PROMO** | 홍보/예고 영상 | WSOP 2025 티저 |

**태깅 원칙**:
- 주요 콘텐츠 유형 1개만 선택
- 복합 유형인 경우 비중이 큰 것 우선
- 불명확한 경우 `기타` 표기 후 메모란에 설명
