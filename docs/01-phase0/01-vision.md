## 용어 정의

> 이 프로젝트에서 사용하는 핵심 용어입니다. **모든 문서에서 아래 용어만 사용합니다.**

### 핵심 개념

| 용어 | 정의 | Phase |
|------|------|:-----:|
| **Single Video** | 한 번에 하나의 비디오만 시청 (기본 OTT) | MVP |
| **Selected View** | 보고 싶은 방송을 선택해서 볼 수 있는 기능 | MVP |
| **Multi-video** | 여러 비디오 플레이어를 동시에 재생하는 기능 | 2+ |
| **Timeshift** | 라이브 중 되감기 기능 | MVP |
| **StatsView** | 플레이어 통계 HUD 오버레이 | 2+ |

### Selected View vs Multi-video (CRITICAL)

| 구분 | Selected View | Multi-video |
|:----:|---------------|-------------|
| **정의** | 어떤 방송을 볼지 **선택** | 여러 비디오를 **동시에** 재생 |
| **구현** | 기본 OTT 기능으로 충분 | 커스텀 플레이어 개발 필요 |
| **예시** | 10개 생방송 중 1개 선택하여 시청 | 4개 테이블을 2x2 그리드로 동시 시청 |
| **Phase** | **MVP** | **2+** |

> **핵심 인사이트**: MVP 단계에서는 Selected View만 구현하면 됨. Multi-video 같은 복잡한 기능 없이 기본 OTT 솔루션으로도 대부분의 핵심 기능 구현 가능.

### Multi-video 구현 방식 (Phase 2+)

| 용어 | 정의 | 예시 |
|------|------|------|
| **Player Cam** | 한 테이블 내 각 선수별 직캠 (아이돌 직캠 방식) | 메인 화면 + 8개 선수 얼굴 캠 |
| **Table Multi-video** | 서로 다른 테이블을 2x2 그리드로 동시 시청 | Main Event + High Roller 동시 시청 |

## 3. Advanced Mode (Phase 2+)

> **참고**: Advanced Mode는 모두 **Multi-video** (여러 비디오 동시 재생) 기반 기능입니다.

### 3.1 Player Cam (Multi-video 구현)

한 테이블 내에서 각 선수별 직캠을 **동시에** 제공하는 기능입니다.

| 구성 요소 | 설명 |
|----------|------|
| **메인 화면** | 테이블 전체 방송 (중앙 배치) |
| **선수 캠** | 각 선수 얼굴 클로즈업 (측면 배치, 동시 재생) |
| **전환** | 선수 클릭 시 해당 캠 확대 |

### 3.2 Table Multi-video (Multi-video 구현)

서로 다른 테이블/대회를 **동시에** 시청하는 기능입니다.

| 구성 요소 | 설명 |
|----------|------|
| **레이아웃** | 2x2 그리드 (최대 4개 비디오 동시 재생) |
| **오디오** | 선택한 테이블 오디오만 재생 |
| **전환** | 테이블 클릭 시 전체 화면 |

### 3.3 StatsView

플레이어 통계를 영상 위에 오버레이로 표시합니다.

| 통계 | 설명 |
|------|------|
| **VPIP** | Voluntarily Put $ In Pot |
| **PFR** | Pre-Flop Raise % |
| **Pot Odds** | 실시간 팟 오즈 |
| **Win Rate** | 예상 승률 |

## 4. 핸드 검색 기능 (Phase 2+)

### 4.1 필수 태깅 항목

| 레벨 | 태깅 데이터 |
|------|------------|
| **Hand 단위** | 핸드 넘버, 시작/종료 타임스탬프 |
| **참여 플레이어** | 해당 핸드에 참여한 선수 목록 |
| **각 플레이어 Hands** | 홀카드 정보 (공개 시점 기준) |
| **Community Card** | 플랍/턴/리버 카드 |
| **최종 Winner** | 해당 핸드 승자 |

### 4.2 검색 예시

- A 선수와 B 선수가 함께 했던 대회/핸드 검색
- 특정 핸드(AA, KK 등)로 이기거나 진 상황 검색
- 포카드 vs 로열 스트레이트 플러시 등 레어 핸드 검색
