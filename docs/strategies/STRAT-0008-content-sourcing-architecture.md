## 1. 개요

WSOP TV 플랫폼의 콘텐츠 소싱 구조를 정의합니다. 모든 WSOP 공식 대회 콘텐츠를 프로덕션 파트너로부터 수집하여 GG Production에서 종편 작업 후 배포합니다.

### 1.1 범위 정의

| 구분 | 포함 | 제외 |
|------|------|------|
| **WSOP 공식** | WSOP Vegas, WSOPE, WSOP Paradise, Super Circuit | - |
| **예외 콘텐츠** | - | Poker After Dark, Game of Gold 등 |

> **원칙**: WSOP 브랜드 공식 대회만 포함. 별도 프로그램은 제외.

---

## 2. 대회 Tier 구조

### 2.1 3-Tier 시스템

| Tier | 대회 | 위치 | 특징 | 연간 이벤트 수 |
|:----:|------|------|------|:--------------:|
| **1** | WSOP Vegas | 라스베가스, 미국 | 메인 시리즈, Main Event 포함 | ~100개 |
| **2** | WSOPE | 유럽 (로젠달, 프라하 등) | 유럽 시리즈 | ~15개 |
| **2** | WSOP Paradise | 바하마 | 파라다이스 시리즈 | ~15개 |
| **3** | Super Circuit | 온라인 | 온라인 시리즈 | ~50개 |

### 2.2 대회별 이벤트 구조

```
WSOP Vegas (Tier 1)
├── Main Event ($10,000 No-Limit Hold'em World Championship)
├── Tournament of Champions
├── $100,000 High Roller
├── $50,000 Poker Players Championship
├── Seniors Event
├── Ladies Event
├── Tag Team Event
└── ... (약 100개 이벤트)

WSOPE (Tier 2)
├── Main Event
├── High Roller
├── Mixed Games
└── ... (약 15개 이벤트)

WSOP Paradise (Tier 2)
├── Main Event
├── High Roller
├── Short Deck
└── ... (약 15개 이벤트)

Super Circuit (Tier 3)
├── Main Event
├── High Roller
├── Mini Main Event
└── ... (약 50개 이벤트)
```

## 3. 프로덕션 파트너

### 3.1 파트너별 담당 영역

| 파트너 | 담당 대회 | 담당 이벤트 | 소스 수령 방식 |
|--------|----------|------------|---------------|
| **ESPN** | WSOP Vegas | Main Event만 | 1년 후 WSOP TV 이관 |
| **PokerGO** | WSOP Vegas | Main Event 외 전체 | 계약에 따라 이관 |
| **Triton Poker** | WSOP Paradise | 전체 이벤트 | 직접 수령 |
| **PokerCaster** | WSOPE | 전체 이벤트 (유럽 지역) | 직접 수령 |

### 3.2 ESPN 상세

| 항목 | 내용 |
|------|------|
| **담당** | WSOP Vegas Main Event |
| **방영** | ESPN 자체 방영 (독점) |
| **이관** | 1년 후 WSOP TV로 이관 |
| **협상** | 계약 조건에 따라 조기 이관 가능 |

### 3.3 PokerGO 상세

| 항목 | 내용 |
|------|------|
| **담당** | WSOP Vegas (Main Event 제외 전체) |
| **방영** | PokerGO 자체 방영 |
| **이관** | 계약 조건에 따라 WSOP TV로 이관 |
| **이벤트 예시** | $100K High Roller, Tournament of Champions, $50K PPC |

### 3.4 Triton Poker 상세

| 항목 | 내용 |
|------|------|
| **담당** | WSOP Paradise 전체 |
| **위치** | 바하마 |
| **수령** | 직접 소스 수령 |

### 3.5 PokerCaster 상세

| 항목 | 내용 |
|------|------|
| **담당** | WSOPE 및 유럽 지역 대회 전체 |
| **위치** | 로젠달, 프라하 등 유럽 |
| **수령** | 직접 소스 수령 |

### 4.2 GG Production 종편 작업

| 작업 | 설명 |
|------|------|
| **소스 수령** | 프로덕션 파트너로부터 원본 영상 수령 |
| **품질 검수** | 해상도, 오디오, 프레임 레이트 확인 |
| **자막 작업** | 20개국 다국어 자막 제작 |
| **그래픽 오버레이** | POT, BOARD, HUD 오버레이 추가 |
| **인코딩** | HLS, DASH 포맷 변환 |
| **핸드 태깅** | 핸드별 메타데이터 입력 (검색용) |

## 5. 연간 콘텐츠 수량

| Tier | 대회 | 방송 이벤트 수 | 비고 |
|:----:|------|:--------------:|------|
| 1 | WSOP Vegas | ~20개 | ESPN 10개 + PokerGO 10개 |

| | **합계** | **~50개** | |
