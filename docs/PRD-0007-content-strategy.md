## Executive Summary

WSOP TV의 핵심 가치 제안인 **콘텐츠 소싱 및 배포 전략**을 정의합니다. 연간 200개 이상의 Bracelet 대회 중 약 50개 방송을 우선 제공하되, 파트너 독점권과 자체 직접 방송을 통해 다층적 콘텐츠 전략을 구축합니다. 추가로 WSOP 역사 전체(1973년~현재)의 아카이브 영상을 메타데이터 정리 하여 제공함으로써 포커 팬의 학습과 오락을 동시에 충족합니다.

### 핵심 목표
- WSOP 모든 대회의 체계적인 콘텐츠 수급 체계 확립
- 파트너 독점권과의 균형을 통한 신뢰 구축
- 역대 영상 아카이브 완성으로 콘텐츠 차별성 확보

## Content Landscape

### 연간 대회 분포

```
연간 약 200개+ Bracelet 대회
│
├── WSOP VEGAS (Las Vegas)
│   ├── 대회 수: ~150개
│   ├── 시즌: 5월~7월 집중 (8-9주)
│   ├── 특징: 동시에 여러 브레이슬릿 진행
│   │         → 전체 방송 불가능 (선택 필요)
│   └── 구성:
│       ├── Main Event (1개)
│       ├── High Roller 시리즈 (15-20개)
│       ├── Mid-Stake 대회 (50-70개)
│       └── Low-Stake 대회 (60-80개)
│
└── 해외 이벤트 (약 50개)
    ├── WSOP Europe (매년 10월, ~30개)
    │   └── 런던 개최
    ├── WSOP Asia-Pacific (불규칙, ~10개)
    │   └── 마닐라, 멜버른 등
    └── WSOP Super Circuit (~10개)
        └── 파트너 카지노 개최
```

### "50개 방송" 정의

> **50개 방송** = 동시 진행으로 **물리적으로 방송할 수 없는 대회**
>
> **NOT**: 나머지 150개를 방송하지 않겠다는 뜻이 아님
>
> **의미**: 우선순위로 50개를 선정하되, 추가 콘텐츠는 가능하면 제공

## Broadcast Strategy

### Phase 1: 우선순위 콘텐츠 (50개)

#### 1.1 WSOP VEGAS 프리미엄 (약 35개)

| 대회 유형 | 수량 | 우선순위 | 이유 |
|---------|------|:-------:|------|
| **Main Event** | 1 | P0 (1위) | WSOP 최고 권위 |
| **High Roller** | 10-12 | P0 | 탑 플레이어, 높은 뷰십 |
| **Mid-Stake** | 15-20 | P1 | 대중적 관심, 접근성 |
| **특별 이벤트** | 5 | P1 | 혼성, 팀 경기 등 |

#### 1.2 해외 이벤트 (약 15개)

| 이벤트 | 수량 | 우선순위 | 설명 |
|-------|------|:-------:|------|
| WSOP Europe Main | 1 | P0 | 유럽 최대 이벤트 |
| WSOP Europe High Roller | 3-5 | P1 | 프리미엄 콘텐츠 |
| WSOP Europe 기타 | 5-7 | P2 | 보조 콘텐츠 |
| WSOP Asia-Pacific | 3-5 | P2 | 아시아 시장 |

### Phase 2: 추가 콘텐츠 (150개+)

#### 2.1 방송 가능 대회

방송 조건을 만족하는 대회는 **가급적 모두 제공**:
- 충분한 제작 인력/장비 확보
- 스트리밍 트래픽 허용
- 파트너 독점 기간 만료

#### 2.2 제외 기준

다음의 경우 제외 가능:

| 제외 사유 | 대회 예시 |
|---------|---------|
| **가치 낮은 콘텐츠** | 주말 토너먼트, Omaha Hi-Lo 등 |
| **높은 제작 비용** | 카메라 3개 이상, 특수 편성 필요 |
| **낮은 뷰십** | 사용자 데이터 기반 판단 |
| **라이선스 불가** | 파트너 독점 기간 중 |

## Partnership Model

### 파트너별 독점 범위 및 기간

| 파트너 | 콘텐츠 범위 | 수량 | 독점 기간 | 동의 |
|-------|-----------|------|---------|------|
| **ESPN** | WSOP VEGAS 프리미엄 (Main + 선정 High Roller) | 10개 | 방송 후 1년 | ✓ 공식 |
| **PokerGo** | WSOP VEGAS High Roller (계약 대상) | 10개 | 계약 기간 | ✓ 기 체결 |
| **WSOP 직접** | 전체 브레이슬릿 (파트너 제외) | 30개+ | 없음 | - |
| **YouTube** | WSOP 직접 방송 라이브 & 재송출 | - | 종료 시 비공개 | ✓ 신규 |

### 독점 기간 상세

#### ESPN 패턴
```
대회 방송 → ESPN 독점 1년 → WSOP TV 제공
(예: 2025 Main Event → ESPN 방송 → 2026년부터 WSOP TV)
```

#### PokerGo 패턴
```
대회 방송 → PokerGo 독점 (계약 기간) → WSOP TV 제공 (가능 시)
(예: High Roller #1 → PokerGo 독점 6개월 → 이후 가능)
```

#### WSOP 직접 패턴
```
대회 방송 → WSOP TV 동시 제공 (독점 기간 없음)
```

## Archive Strategy

### 역대 영상 수집 범위

#### 4.1 시간 범위
- **시작**: 1973년 (WSOP 창립)
- **끝**: 현재
- **총 기간**: 50년+ 콘텐츠

#### 4.2 대회 유형
```
1973년 ~ 현재
│
├── WSOP Main Event (1974년부터 매년)
│   └── 약 50개 영상
│
├── WSOP High Roller 시리즈 (2009년부터 본격)
│   └── 약 150개+ 영상
│
├── 공식 캐시 게임 (2000년대)
│   └── 약 30개 영상
│
└── WSOP Europe (2007년부터)
    └── 약 60개+ 영상
```

### 우선순위

| 우선순위 | 대회 | 수량 | 비고 |
|:-------:|-----|------|------|
| **P0** | Main Event | 50 | 역사적 의미 최고 |
| **P1** | High Roller | 80 | 엘리트 플레이어 |
| **P2** | 주요 Bracelet | 100 | 대중적 인기 |
| **P3** | 기타 | 200+ | 완성도 |

### 메타데이터 정리

#### 4.3 필수 메타데이터

```yaml
대회_정보:
  - 년도 (YYYY)
  - 번호 (Event #)
  - 이름 (Tournament Name)
  - 게임 타입 (Hold'em, Omaha, etc.)
  - Buy-in ($)
  - 개최지 (Vegas, Europe, etc.)

결과_정보:
  - 우승자 (Winner Name)
  - 우승상금 (Prize)
  - 참가자 수 (Field Size)
  - Final Table 테이블 수

영상_정보:
  - 방송 채널 (ESPN, WSOP 원본, etc.)
  - 영상 길이 (Duration)
  - 화질 (Resolution)
  - 해설자 (Commentators)
  - Final Table 영상 (Hours)
  - 내용 설명 (Description)

액세스_정보:
  - 저장 위치 (Storage Path)
  - CDN URL (if available)
  - 라이센스 상태 (License Status)
```

#### 4.4 메타데이터 자동화

```
영상 인수 → OCR (자동 추출)
          ↓
        수동 검증 (QA)
          ↓
        DB 저장 + CDN 업로드
          ↓
        검색 인덱싱 (Elasticsearch)
```

## Technical Implementation

### Content Ingestion Pipeline

```
┌──────────────┐
│ 대회 방송     │
│ (Production) │
└───────┬──────┘
        │
        ▼
┌──────────────────────┐
│ 1. Raw Video Capture │
│ (RTMP/HLS)          │
└──────────┬───────────┘
           │
           ▼
┌──────────────────────┐
│ 2. Format Normalize  │
│ (Codec, Resolution)  │
└──────────┬───────────┘
           │
           ▼
┌──────────────────────┐
│ 3. Metadata Extract  │
│ (OCR, Manual Entry)  │
└──────────┬───────────┘
           │
           ▼
┌──────────────────────┐
│ 4. Cloud Storage     │
│ (S3/GCS)            │
└──────────┬───────────┘
           │
           ▼
┌──────────────────────┐
│ 5. CDN Distribution  │
│ (CloudFront/Akamai)  │
└──────────┬───────────┘
           │
           ▼
┌──────────────────────┐
│ 6. Search Indexing   │
│ (Elasticsearch)      │
└──────────┬───────────┘
           │
           ▼
┌──────────────────────┐
│ 7. Streaming Delivery│
│ (WSOP TV/YouTube)   │
└──────────────────────┘
```

### API & Database

#### 콘텐츠 메타데이터 저장소
```
PostgreSQL (구조화된 데이터)
├── tournaments (대회 정보)
├── broadcasts (방송 기록)
├── metadata (메타데이터)
└── licenses (라이센스 추적)

Elasticsearch (전문 검색)
├── 빠른 검색 인덱싱
├── 필터링 (년도, 게임 타입 등)
└── 추천 알고리즘 연동
```
