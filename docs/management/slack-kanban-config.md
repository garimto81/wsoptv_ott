# Slack Kanban Board 설정

**생성일**: 2026-02-02
**Workspace**: GGProduction

## 사용 방법

1. **Slack 열기**
2. 좌측 사이드바 → **Lists** 탭 클릭
3. **"WSOPTV 업체 관리 칸반보드"** 선택
4. 우측 상단 필터 아이콘 → **Layout** → **Board** 선택

## 칸반 컬럼 (상태)

| 상태 | 의미 |
|------|------|
| **RFP 발송** | 제안 요청 발송 완료 |
| **견적 대기** | 견적서 회신 대기 중 |
| **평가 중** | 평가표 작성 중 |
| **협상 중** | 우선협상대상 선정, 계약 협상 |
| **완료** | 계약 체결 완료 |
| **제외** | 선정 불가 |

## 초기 업체 데이터

| 업체 | 상태 | 견적 | 비고 |
|------|------|------|------|
| 메가존클라우드 | 협상 중 | 48억 + 15억/년 | 최우선 협상 대상 |
| Brightcove | 견적 대기 | - | D+5 미회신 |
